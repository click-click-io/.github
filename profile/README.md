## 🧹 About Clean-Click

**Clean-Click** is a comprehensive SaaS platform designed specifically for the **building cleaning industry** (_Gebäudereinigung_). We help cleaning companies streamline their operations, improve communication, and ensure quality control — all without the chaos of WhatsApp groups or expensive, complicated tools.

### 🎯 Our Mission

> *Schluss mit Chaos in WhatsApp-Chats! Schluss mit überteuerten, komplizierten Tools! Schluss mit Kundenbeschwerden und Unzuverlässigkeit!*

We provide cleaning companies with **clear communication without language barriers**, **simple tools with exactly the features they need**, and a system that ensures **reliable deployments and satisfied customers**.

---

## ✨ Key Features

### 📱 Mobile App for Field Staff
- **Task Management** — Clear, visual task assignments for cleaning staff
- **Time Tracking** — GPS-enabled clock-in/clock-out with location verification
- **Photo Documentation** — Upload before/after photos for quality assurance
- **Offline Support** — Works even without internet connection
- **Multi-language Support** — Interface available in German, English, Turkish, and Arabic

### 🖥️ Management Portal
- **Dashboard & Analytics** — Real-time insights into operations and performance
- **Employee Management** — Complete staff administration and scheduling
- **Customer Management** — CRM for managing client relationships
- **Location Management** — Organize and track all cleaning locations
- **Shift Scheduling** — Visual calendar for planning and assignment
- **Job & Project Management** — Create and track cleaning jobs and recurring projects
- **Quality Reports** — Generate detailed shift reports with photo evidence
- **Real-time Chat** — Built-in messaging for instant team communication

### 🔔 Smart Notifications
- **Push Notifications** — Instant alerts for new assignments and updates
- **SMS Notifications** — Critical alerts via text message
- **Email Notifications** — Automated reports and summaries

### 🔐 Security & Compliance
- **Role-based Access Control** — Admin, Manager, and Employee roles
- **Secure Authentication** — JWT-based authentication
- **GDPR Compliant** — Data handling in compliance with European regulations
- **Data hosted in Germany** — Your data stays protected under German law

## 🏗️ Platform Architecture

Clean-Click consists of multiple interconnected applications:

| Component | Description | Tech Stack |
|-----------|-------------|------------|
| **Mobile App** | Progressive Web App for cleaning staff | React, TypeScript, Vite, TailwindCSS |
| **Company Portal** | Web dashboard for management & admins | React, TypeScript, MUI, Vite |
| **Admin Portal** | Internal administration dashboard | React, TypeScript, MUI, Vite |
| **Backend API** | RESTful API with business logic | Node.js, Express, MongoDB, TypeScript |
| **Socket Server** | Real-time communication service | Node.js, Socket.io, TypeScript |
